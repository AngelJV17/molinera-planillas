# MOLICENTE

MOLICENTE es un sistema web desarrollado para apoyar la gestión administrativa de la empresa Industria Molinera San Vicente S.R.L., especialmente en los procesos relacionados con el personal, la asistencia mensual, la generación de planillas y la emisión de boletas de pago.

El sistema busca reducir el uso de registros manuales, organizar mejor la información laboral y facilitar el seguimiento de los procesos internos de la empresa.

## Descripción general

El proyecto permite administrar la información de los trabajadores, registrar su asistencia mensual, controlar faltas, descansos, canjes y horas extras. Además, facilita la generación de planillas y boletas de pago, permitiendo que la información esté centralizada y disponible para los responsables de cada proceso.

MOLICENTE está orientado a mejorar la eficiencia, el orden y la trazabilidad de la gestión laboral dentro de la empresa.

## Funcionalidades principales

- Gestión de trabajadores.
- Registro de asistencia mensual.
- Control de faltas, descansos y canjes.
- Cálculo de horas trabajadas y horas extras.
- Generación y revisión de planillas.
- Emisión de boletas de pago.
- Administración de catálogos y parámetros generales.
- Gestión de usuarios, roles y permisos.
- Consulta de reportes administrativos.

## Tecnologías utilizadas

El sistema fue desarrollado con tecnologías web modernas, utilizando Laravel para el backend, Vue 3 e Inertia.js para la interfaz, Tailwind CSS para el diseño y MySQL como gestor de base de datos.

## Finalidad del proyecto

La finalidad de MOLICENTE es brindar una solución digital que permita mejorar el control del personal y optimizar los procesos de asistencia, planillas y boletas dentro de la organización.
